Introduction

The Python programming language is an object-oriented language, which means that it can model real-world entities. It is also dynamically-typed because it carries out type-checking at runtime. It does so to make sure that the type of construct matches what we expect it to be. The distinctive feature of Python is that it is an interpreted language. The Python IDLE (Integrated Development Environment) executes instructions one line at a time.

The Python Interpreter

Python interpreter offers some pretty cool features:

  • Interactive editing.
  • History substitution.
  • Code completion on systems with support for readline.

On Windows, when you want to run the Python interpreter in the shell, you can type the following: >>> $python
>>>     # start writing your code here To get out of the interpreter in disassembling the Bytecode shell, you can type: >>> exit()

How Does Python Interpreter Work?

Well, internally, four things happen in a REPL:

  • Lexing- The lexer breaks the line of code into tokens.
  • Parsing- The parser uses these tokens to generate a structure, here, an Abstract Syntax Tree, to depict the relationship between these tokens.
  • Compiling- The compiler turns this AST into code object(s).
  • Interpreting- The interpreter executes each code object.

Variables

What are Python Variables?

A variable is a container for a value. It can be assigned a name, you can use it to refer to it later in the program. Based on the value assigned, the interpreter decides its data type. You can always store a different type in a variable. To assign a value to Python variables, you don’t need to declare its type. You name and type the value after the equal sign(=): >>> num = 1       # an int variable
>>> name = "Cat"     # a string variable
>>> checked = True     # a bool variable

To display a single character from a string, put its index in square brackets. Indexing begins at 0. >>> s = "I love Python."

If Statement

We’ll start by looking at the most basic type of if statement. In its simplest form, it looks like this: if it rains:  # expression

  • If [expression] evaluated in Boolean contextion, [statement] is a valid Python statement, which must be indented.
  • If [expression] is true (evaluates to a value that is “truthy”), then [statement] is executed.
  • If [expression] is false, then [statement] is skipped over and not executed.

Note that the colon (:) following [expression] is required. Some programming languages require [expression] to be enclosed in parentheses, but Python does not. Here are several examples of this type of if statement: >>> x = 0

Lists

A list in Python is a collection of arbitrary objects, somewhat akin to an array in many other programming languages but more flexible. Lists are defined in Python by enclosing a comma-separated sequence of objects in square brackets ([]), as shown below: >>> a = ['foo', 'bar', 'baz', 'qux']
>>> print(a)
['foo', 'bar', 'baz', 'qux']     # output
>>> a
['foo', 'bar', 'baz', 'qux']     # output The important characteristics of Python lists are as follows:

  • Lists are ordered.
    • >>> a = ['foo', 'bar', 'baz', 'qux']
    >>> b = ['baz', 'qux', 'bar', 'foo']
    >>> a == b
    False     # output
    >>> a is b
    False     # output
    >>> [1, 2, 3, 4] == [4, 1, 3, 2] False     # output
  • Lists can contain any arbitrary objects. >>> a = [21.42, 'foobar', 3, 4, 'bark', False, 3.14159]
    >>> a
    [21.42, 'foobar', 3, 4, 'bark', False, 3.14159]     # output
  • Lists elements can be accessed by index. >>> a = ['foo', 'bar', 'baz', 'qux', 'quux', 'corge'] >>> a[0]
    'foo'
    >>> a[2]
    'baz'
    >>> a[5]
    'corge'
  • Lists can be nested to arbitrary depth. >>> x = ['a', ['bb', ['ccc', 'ddd'], 'ee', 'ff'], 'g', ['hh', 'ii'], 'j']
    >>> x
    ['a', ['bb', ['ccc', 'ddd'], 'ee', 'ff'], 'g', ['hh', 'ii'], 'j']     # output
  • Lists are mutable. >>> a = ['foo', 'bar', 'baz', 'qux', 'quux', 'corge']
    >>> a
    ['foo', 'bar', 'baz', 'qux', 'quux', 'corge']     # output
    >>> a[2] = 10
    >>> a[-1] = 20
    >>> a
    ['foo', 'bar', 10, 'qux', 'quux', 20]     # output
  • Lists are dynamic. >>> a = ['foo', 'bar', 'baz', 'qux', 'quux', 'corge']
    >>> a += ['grault', 'garply']
    >>> a
    ['foo', 'bar', 'baz', 'qux', 'quux', 'corge', 'grault', 'garply']     # output
    >>> a = ['foo', 'bar', 'baz', 'qux', 'quux', 'corge']
    >>> a = [10, 20] + a
    >>> a
    [10, 20, 'foo', 'bar', 'baz', 'qux', 'quux', 'corge']     # output

Tuples

Tuples are identical to lists in all respects, except for the following properties:

  • Tuples are defined by enclosing the elements in parentheses (()) instead of square brackets ([]).
  • Tuples are immutable.

>>> t = ('foo', 'bar', 'baz', 'qux', 'quux', 'corge')
>>> t
('foo', 'bar', 'baz', 'qux', 'quux', 'corge') >>> t[0]
'foo' >>> t[-1]
'corge' >>> t[1::2]
('bar', 'qux', 'corge')

Loops

When you want some statements to execute a hundred times, you don’t repeat them 100 times. Think of when you want to print numbers 1 to 99. Or that you want to say Hello to 99 friends. In such a case, you can use loops in python. In next sections we'll discuss two of the most import loops in python, while and for loops.

While Loop

A while loop in python iterates till its condition becomes False. In other words, it executes the statements under itself while the condition it takes is True.When the program control reaches the while loop, the condition is checked. If the condition is true, the block of code under it is executed. Remember to indent all statements under the loop equally. After that, the condition is checked again. This continues until the condition becomes false. Then, the first statement, if any, after the loop is executed. >>> a= 3
>>> while(a>0):
...     print(a)
...     a-= 1
3     #output
2    
1    

For Loop

Python for loop can iterate over a sequence of items. The structure of a for loop in Python is different than that in C++ or Java. That is, for(int i=0; i< n;i++) won’t work here. In Python, we use the in keyword. >>> a = ['foo', 'bar', 'baz']
>>> for i in a:
... print(i)
...
foo     #output
bar
baz
